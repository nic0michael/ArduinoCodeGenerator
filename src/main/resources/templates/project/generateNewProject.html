Generate New Project From Existing Code

In order to use this facility you need to find one or more Project GUID texts by Listing all projects
or by using Search Projects (searching by category) to find all the Project GUIDs you need.
This will then aggregate all to code into one raw project that you need to masage and add your own code to make your new project work
Once you have a new project that works you can then contribute this as a new Project.
We would recoment to keep all the contributors names as well as yours in the ContributorsName Field as recognition to their original code
